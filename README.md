# Music Recommender System 
Tech stack : Python Django framewowrk, HTML, CSS, Javascript, AJAX, SQLite

## How to run
1. Clone the repository 
2. Navigate to "Django Project" folder (cd "Django Project")
3. Create a Virtual environment (python -m venv djangoVenv)
4. Go to the Virtual enviroment (.\djangoVenv\Scripts\activate.ps1)
5. Navigate to "musicRecommender" folder (cd musicRecommender)
6. Install dependencies from requirements.txt (pip install -r requirements.txt)
7. Run "python manage.py runserver 127.0.0.1:8080"
8. Visit the development server http://127.0.0.1:8080/
